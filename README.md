# Provium — Autonomous DeFi Compliance with Zero Knowledge

> *"The math never lies. And now it proves compliance without betraying your users."*

**Provium** uses three AI agents to autonomously monitor DeFi lending protocols, generate zero-knowledge proofs of collateral compliance, and store cryptographically verifiable reports on-chain — all without exposing individual user positions.

---

## 🚀 Quick Start

See **[DEPLOY.md](./DEPLOY.md)** for complete deployment instructions.

**TL;DR:**
```bash
# Contracts
cd contracts && npm install
cp .env.example .env  # Fill DEPLOYER_PRIVATE_KEY, AGENT_WALLET_ADDRESS
npm run compile-circuit  # OR: npx ts-node scripts/compile-circuit.ts
npx hardhat compile
npm run deploy
npm run seed

# Agent
cd ../agent
python -m venv venv && venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env  # Fill AGENT_PRIVATE_KEY
python main.py --once

# Dashboard
cd ../dashboard
npm install
cp .env.local.example .env.local  # Fill addresses from base-sepolia.json
npm run dev
```

## 🎯 How It Works

1. **Watch** — Watcher agent monitors all positions on-chain, assesses risk level (`low / medium / high / critical`)
2. **Analyze** — Analyst agent (Groq `llama-3.3-70b-versatile`) decides what proofs to generate, writes on-chain reasoning
3. **Prove** — Reporter agent builds Merkle tree, runs Noir ZK circuit via Barretenberg (nargo + bb)
4. **Deliver** — Proof cryptographically verified by `UltraVerifier.sol` and stored immutably on Base Sepolia

**Key innovation:** LLM reasoning is stored on-chain forever. Regulators can verify compliance cryptographically without seeing individual user data.
